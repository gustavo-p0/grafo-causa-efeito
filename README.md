# Grafo de Causa e Efeito

Material interativo em React para estudo de Grafo de Causa e Efeito, Tabelas de Decisão e geração de casos de teste.

## Scripts

Depois de instalar as dependências (`npm install`):

- `npm run dev` — sobe o servidor de desenvolvimento
- `npm run build` — build de produção
- `npm run preview` — pré-visualiza o build
